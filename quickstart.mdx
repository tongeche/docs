## Use this documentation by audience

### Riders and general users

1. Start on [TIBS documentation](/).
2. Open [Product overview](/product/overview).
3. Go to [Legal center](/legal/index) if you need privacy or policy
   information.

### Surveyors and program participants

1. Start with [Product overview](/product/overview).
2. Open [Surveyor](/product/surveyor) for the fieldwork overview.
3. Review the [Surveyor participation agreement](/legal/surveyor-participation-agreement)
   when relevant.

### Partners and integrators

1. Read [API introduction](/api-reference/introduction).
2. Use [Live interfaces](/api-reference/live-interfaces) to understand the
   current system.
3. Use [Future public API draft](/api-reference/future-public-api) if you need
   the planned external contract shape.

### Legal and policy readers

1. Start in [Legal center](/legal/index).
2. Review the privacy policy, reporting policy, and any agreement relevant to
   your use of TIBS.

### Documentation contributors

1. Read [Development](/development).
2. Preview changes locally.
3. Validate links before publishing updates.

## What this site includes

- product guidance for TIBS services
- legal and policy information
- technical reference material for developers and integrators
