## Current product surfaces

The current documentation scaffold is based on the endpoint-to-table mapping in
`references/tibs_ref.md`. The main TIBS domains already identified are:

- Reports
- Incidents
- Vehicles
- Feedback
- Search
- Corridors, routes, and saccos
- Surveyor operations
- Learning
- Restricted admin flows
