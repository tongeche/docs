## Purpose

`tibs-survey` is a separate application for surveyor operations and project
management. It handles invite-only access, project-scoped workflows, survey
submissions, assignments, expense claims, payout operations, and internal admin
review.

## Stack

- Next.js 16 App Router
- React 19
- Tailwind CSS v4
- pnpm
- Supabase Auth and Postgres
- Netlify
- Brevo transactional email
- M-Pesa payout integration

## Current feature surface

The current route map includes:

- auth: login, signup, forgot password, invite acceptance, auth callback
- onboarding and access control
- dashboard and assignments
- survey submission and survey history
- project dashboards under `/projects/[projectSlug]`
- expenses, payouts, invoices, and payout invoice verification
- documents and field documents
- admin pages for surveyors, vehicles, payouts, slots, and approvals
- public privacy policy page

## Environment and local setup

From the repo root:

```bash
pnpm install
pnpm dev
```

Key env groups visible in `.env.example`:

- Supabase public and secret keys
- public origin and hosted-origin overrides
- shared learning-center URL
- admin email allowlist
- Brevo email configuration
- M-Pesa payout configuration

## Deployment

- Build command: `pnpm build`
- Netlify config: `netlify.toml`
- Netlify pins Node `22`
- Scheduled Netlify functions exist for inactivity reminders and founders onboarding
